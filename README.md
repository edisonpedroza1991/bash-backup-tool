# Backup Script

## Description
A Bash script that creates compressed, timestamped backups of a directory, with built-in error handling.

## Features
- Compressed backups (`.tar.gz`)
- Timestamped filenames (date and time)
- Error handling if the source folder is missing
- Automatic creation of the destination folder

## Usage
\`\`\`bash
chmod +x backup.sh
./backup.sh
\`\`\`

## Requirements
- Bash
- A Unix-based system (Linux or macOS)
